# Relebrando comandos git/Github

## configuraçoes
Adicione o nome do desenlvovedor:
```bash
git config --global user.name "Rodrigo A. Lima dos Santos"
```
Adiciona o e-mail do desenvolvedor:
```bash
git config --global user.email rodrigosantos5510jau@gmail.com
```

## comando git 
Rode apenas uma vez para executar o git dentro da pasta
```bash
git config --global user.email rodrigosantos5510jau@gmail.com
```
Cria uma copia dos arquivos para subir em nuvem
```bash
git add .
```
verifica a atual situção dos arquivos
```bash
git status
```

sobe o comentario sibre atualização dos arquivos 
```bash
git commit -m "comentario"
```

sobe os arquivos para o repositorio
```bash
git push
```
